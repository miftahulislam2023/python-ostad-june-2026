# String Methods in Python
- পাইথনে **String (স্ট্রিং)** হচ্ছে এক বা একাধিক ক্যারেক্টারের সমষ্টি, যা সিঙ্গেল কোট (`'...'`) বা ডাবল কোট (`"..."`) এর মধ্যে লেখা হয়। পাইথনে স্ট্রিং ম্যানিপুলেট বা পরিবর্তন করার জন্য অনেকগুলো বিল্ট-ইন মেথড (Built-in Methods) আছে।

## ১. কেস পরিবর্তন করার মেথড (Case Conversion Methods)

এই মেথডগুলো স্ট্রিংয়ের অক্ষরগুলোকে বড় হাতের (Uppercase) বা ছোট হাতের (Lowercase) অক্ষরে রূপান্তর করতে ব্যবহৃত হয়।

* **`upper()`**: স্ট্রিংয়ের সব অক্ষরকে বড় হাতের অক্ষরে রূপান্তর করে।
* **`lower()`**: স্ট্রিংয়ের সব অক্ষরকে ছোট হাতের অক্ষরে রূপান্তর করে।
* **`capitalize()`**: স্ট্রিংয়ের শুধুমাত্র প্রথম অক্ষরটিকে বড় হাতের করে এবং বাকিগুলো ছোট হাতের করে।
* **`title()`**: স্ট্রিংয়ের প্রতিটি শব্দের প্রথম অক্ষরটিকে বড় হাতের করে।
* **`swapcase()`**: বড় হাতের অক্ষরকে ছোট হাতের এবং ছোট হাতের অক্ষরকে বড় হাতের অক্ষরে রূপান্তর করে।

### উদাহরণ:

```python
text = "hello WORLD"

print(text.upper())       # আউটপুট: HELLO WORLD
print(text.lower())       # আউটপুট: hello world
print(text.capitalize())  # আউটপুট: Hello world
print(text.title())       # আউটপুট: Hello World
print(text.swapcase())    # আউটপুট: HELLO world

```

## ২. অনুসন্ধান এবং গণনা করার মেথড (Searching & Counting Methods)

স্ট্রিংয়ের ভেতরে কোনো নির্দিষ্ট অংশ বা ক্যারেক্টার খোঁজার জন্য এগুলো ব্যবহার করা হয়।

* **`count(sub)`**: একটি নির্দিষ্ট সাব-স্ট্রিং মূল স্ট্রিংয়ে কতবার আছে তা গণনা করে।
* **`find(sub)`**: নির্দিষ্ট সাব-স্ট্রিংটি প্রথম কোথায় আছে তার ইনডেক্স (Index) নম্বর দেয়। খুঁজে না পেলে `-1` রিটার্ন করে।
* **`index(sub)`**: `find()` এর মতোই কাজ করে, তবে সাব-স্ট্রিংটি খুঁজে না পেলে এটি একটি এরর (ValueError) দেয়।
* **`startswith(prefix)`**: স্ট্রিংটি নির্দিষ্ট কোনো শব্দ বা অক্ষর দিয়ে শুরু হয়েছে কিনা তা পরীক্ষা করে (True/False দেয়)।
* **`endswith(suffix)`**: স্ট্রিংটি নির্দিষ্ট কোনো শব্দ বা অক্ষর দিয়ে শেষ হয়েছে কিনা তা পরীক্ষা করে।

### উদাহরণ:

```python
text = "Python is amazing and Python is easy."

print(text.count("Python"))     # আউটপুট: 2
print(text.find("amazing"))     # আউটপুট: 10 (ইনডেক্স ১০ নম্বর পজিশনে আছে)
print(text.find("Java"))        # আউটপুট: -1 (কারণ Java স্ট্রিংয়ে নেই)
print(text.startswith("Python"))# আউটপুট: True
print(text.endswith("easy."))   # আউটপুট: True

```

## ৩. স্ট্রিং মডিফাই বা পরিবর্তন করার মেথড (Modification Methods)

* **`strip()`**: স্ট্রিংয়ের শুরু এবং শেষের সব হোয়াইটস্পেস (ফাঁকা জায়গা) বাদ দিয়ে দেয়।
* **`lstrip()`**: শুধুমাত্র বাম পাশের (শুরুর) হোয়াইটস্পেস বাদ দেয়।
* **`rstrip()`**: শুধুমাত্র ডান পাশের (শেষের) হোয়াইটস্পেস বাদ দেয়।
* **`replace(old, new)`**: স্ট্রিংয়ের কোনো নির্দিষ্ট অংশকে অন্য কোনো নতুন অংশ দিয়ে প্রতিস্থাপন করে।

### উদাহরণ:

```python
spaced_text = "   Hello Python   "
print(spaced_text.strip())   # আউটপুট: "Hello Python"
print(spaced_text.lstrip())  # আউটপুট: "Hello Python   "

text = "I love Java"
print(text.replace("Java", "Python")) # আউটপুট: I love Python

```

## ৪. স্ট্রিং বিভাজন এবং জোড়া দেওয়ার মেথড (Splitting & Joining Methods)

* **`split(separator)`**: একটি নির্দিষ্ট সেপারেটর (যেমন: স্পেস, কমা) এর ওপর ভিত্তি করে স্ট্রিংটিকে ভেঙে একটি লিস্ট (List)-এ রূপান্তর করে। বাই-ডিফল্ট এটি স্পেস ধরে কাজ করে।
* **`join(iterable)`**: একটি লিস্টের সব উপাদানকে একটি নির্দিষ্ট স্ট্রিংয়ের মাধ্যমে জোড়া লাগিয়ে একটি নতুন স্ট্রিং তৈরি করে।

### উদাহরণ:

```python
# split() এর উদাহরণ
fruits = "apple, banana, cherry"
fruits_list = fruits.split(", ")
print(fruits_list)  # আউটপুট: ['apple', 'banana', 'cherry']

# join() এর উদাহরণ
words = ['Python', 'is', 'fun']
sentence = " ".join(words)
print(sentence)     # আউটপুট: Python is fun

```

## ৫. স্ট্রিং যাচাইকরণ মেথড (Validation Methods)

এই মেথডগুলো স্ট্রিংয়ের ভেতরের ক্যারেক্টারগুলো কেমন তা যাচাই করে **True** অথবা **False** রিটার্ন করে।

* **`isalpha()`**: স্ট্রিংয়ের সবগুলো ক্যারেক্টার যদি অ্যালফাবেট (A-Z, a-z) হয়, তবে True দেয়। (স্পেস থাকলে False দেবে)।
* **`isdigit()`**: স্ট্রিংয়ের সবগুলো ক্যারেক্টার যদি সংখ্যা (0-9) হয়, তবে True দেয়।
* **`isalnum()`**: স্ট্রিংয়ে যদি শুধু অক্ষর এবং সংখ্যা থাকে (কোনো স্পেস বা স্পেশাল ক্যারেক্টার ছাড়া), তবে True দেয়।
* **`islower()`**: সব অক্ষর ছোট হাতের হলে True দেয়।
* **`isupper()`**: সব অক্ষর বড় হাতের হলে True দেয়।
* **`isspace()`**: স্ট্রিংয়ে যদি কেবল হোয়াইটস্পেস বা ফাঁকা জায়গা থাকে, তবে True দেয়।

### উদাহরণ:

```python
print("Python3".isalnum()) # আউটপুট: True (অক্ষর ও সংখ্যা আছে)
print("12345".isdigit())   # আউটপুট: True (সব সংখ্যা)
print("Hello".isalpha())   # আউটপুট: True (সব অক্ষর)
print("Hello World".isalpha()) # আউটপুট: False (মাঝে স্পেস আছে)

```
